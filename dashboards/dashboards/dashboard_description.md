# Banking Client Retention Dashboard

This dashboard analyzes customer churn risk for a retail bank.

## Key Metrics
- Total Customers
- Churn Rate
- High-Risk Customers
- Average Products per Customer

## Visualizations

1. Churn Rate by Geography
Shows which regions have the highest customer churn.

2. Churn by Number of Products
Customers with only 1 product have significantly higher churn risk.

3. Active vs Inactive Members
Inactive members churn at a much higher rate.

4. Risk Band Distribution
Customers categorized into Low, Medium, and High churn risk.

## Tools Used
- SQL for data preparation
- Tableau / Power BI for visualization
